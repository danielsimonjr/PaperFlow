# PaperFlow Keyboard Shortcuts

This document provides a comprehensive list of all keyboard shortcuts available in PaperFlow.

> **Note:** On macOS, use `Cmd` instead of `Ctrl`. The shortcuts below use `Ctrl` but will automatically use `Cmd` on macOS.

| Action | Windows/Linux | macOS |
|--------|---------------|-------|
| New Window | `Ctrl+Shift+N` | `Cmd+Shift+N` |
| Open File | `Ctrl+O` | `Cmd+O` |
| Save | `Ctrl+S` | `Cmd+S` |
| Save As | `Ctrl+Shift+S` | `Cmd+Shift+S` |
| Close Document | `Ctrl+W` | `Cmd+W` |
| Print | `Ctrl+P` | `Cmd+P` |
| Exit/Quit | `Alt+F4` | `Cmd+Q` |

## Customizing Shortcuts

PaperFlow allows you to customize keyboard shortcuts to match your preferences:

1. Open **Preferences** (`Ctrl+,` / `Cmd+,`)
2. Navigate to **Keyboard Shortcuts** section
3. Click on any shortcut to edit it
4. Press your desired key combination
5. Click **Save** to apply the change

### Conflict Detection

If you try to assign a shortcut that's already in use, PaperFlow will warn you about the conflict. You can either:
- Choose a different key combination
- Clear the conflicting shortcut first

### Resetting Shortcuts

To reset a single shortcut:
1. Click on the shortcut
2. Click **Reset to Default**

To reset all shortcuts:
1. Go to **Preferences** > **Keyboard Shortcuts**
2. Click **Reset All to Defaults**

## Platform-Specific Notes

### macOS
- Use `Cmd` instead of `Ctrl` for most shortcuts
- `Option` is used instead of `Alt`
- Full screen shortcut is `Ctrl+Cmd+F` (not `F11`)
- Application menu (`Cmd+,` for Preferences) follows macOS conventions

### Windows
- `Alt+F4` closes the application
- `F11` toggles full screen
- Preferences is in the Edit menu

### Linux
- Shortcuts follow Windows conventions
- Some desktop environments may have conflicting global shortcuts

## Accessibility

PaperFlow supports keyboard-only navigation:
- `Tab` / `Shift+Tab` - Navigate between UI elements
- `Enter` / `Space` - Activate buttons and controls
- `Escape` - Close dialogs and menus
- Arrow keys - Navigate within menus and lists

## Tips

1. **Quick Zoom**: Use `Ctrl+0` to reset zoom to 100% if you lose track of the zoom level
2. **Fast Navigation**: Use `Home`/`End` keys to jump to the start/end of the document
3. **Annotation Mode**: Press the annotation shortcut key twice to toggle off the tool
4. **Multi-Select**: Hold `Shift` while clicking to select multiple annotations
