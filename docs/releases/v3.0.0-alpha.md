# PaperFlow v3.0.0-alpha.1 Release Notes

**Release Date:** February 2, 2026

**Release Type:** Alpha (Pre-release)

This is the first alpha release of PaperFlow Desktop, introducing native desktop functionality while retaining all the features of the web-based PWA.

## Highlights

### Native Desktop Experience

PaperFlow is now available as a native desktop application for Windows, macOS, and Linux. This brings significant improvements:

- **Native Performance**: Direct file system access without browser sandboxing
- **System Integration**: File associations, protocol handlers, and OS-level features
- **Offline First**: Work entirely offline without an internet connection
- **Auto-Updates**: Receive updates automatically without manual downloads

### Key Features

#### Native File Operations
- Open PDFs directly from Explorer/Finder with file associations
- Save files instantly with native save dialogs
- Auto-save and crash recovery to protect your work
- File watching for external change detection

#### System Integration
- System tray with quick access to recent files
- Native notifications for save confirmations and batch operations
- Print directly to system printers
- Deep linking with `paperflow://` URLs

#### Cross-Platform Support
- **Windows**: NSIS installer and portable version
- **macOS**: DMG with notarization for Gatekeeper
- **Linux**: AppImage and .deb packages

## Installation

### Windows

1. Download `PaperFlow-Setup-3.0.0-alpha.1.exe`
2. Double-click to run the installer
3. Follow the installation wizard
4. PaperFlow will be available in the Start Menu

### macOS

1. Download `PaperFlow-3.0.0-alpha.1.dmg`
2. Open the DMG file
3. Drag PaperFlow to the Applications folder
4. Open PaperFlow from Applications
5. If prompted by Gatekeeper, click "Open Anyway" in System Preferences > Security

### Linux

**AppImage:**
```bash
chmod +x PaperFlow-3.0.0-alpha.1.AppImage
./PaperFlow-3.0.0-alpha.1.AppImage
```

**Debian/Ubuntu:**
```bash
sudo dpkg -i paperflow_3.0.0-alpha.1_amd64.deb
```

## New Features in Detail

### Native Print Dialog
- Access all system printers
- Set copies, page ranges, color/grayscale
- Preview before printing
- PDF-specific print settings

### Protocol Handler
Open files from links using `paperflow://open?file=/path/to/file.pdf`

### Keyboard Shortcuts
All shortcuts work natively, including:
- `Ctrl/Cmd + O` - Open file
- `Ctrl/Cmd + S` - Save file
- `Ctrl/Cmd + P` - Print
- `Ctrl/Cmd + +/-` - Zoom in/out

### System Tray
- Minimize to tray instead of taskbar
- Quick access to recent files
- Notification badges
- Right-click menu

### Auto-Update
- Automatic update checking
- Download in background
- Install on next launch or immediately
- Update channel selection (stable/beta/alpha)

## Known Issues

1. **Linux File Associations**: May require manual configuration on some distributions
2. **Wayland System Tray**: Requires AppIndicator extension on GNOME
3. **Large PDFs**: Documents over 500 pages may use significant memory
4. **OCR**: Processing time varies based on document complexity

## Breaking Changes

- This is a new major version with different architecture
- Settings from the PWA version are preserved but may need reconfiguration
- Recent files need to be re-opened once
