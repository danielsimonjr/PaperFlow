## What is PaperFlow?

PaperFlow is a Progressive Web Application (PWA) for viewing, editing, and annotating PDF documents. Unlike traditional PDF editors, PaperFlow:

- **Runs entirely in your browser** - No software to install
- **Works offline** - Install as a PWA for offline access
- **Respects your privacy** - All processing happens locally on your device
- **Is completely free** - No subscriptions or hidden fees

## Key Features

### View & Navigate
- Open PDFs by clicking or drag-and-drop
- Zoom in/out with controls or pinch-to-zoom
- Navigate with thumbnails or page numbers
- Search text within documents

### Annotate
- Highlight, underline, or strikethrough text
- Add sticky notes with comments
- Draw freehand or use shape tools
- Add stamps and custom images

### Fill Forms
- Detect and fill interactive form fields
- Support for text, checkboxes, radio buttons, dropdowns
- Export form data as FDF/XFDF

### Sign Documents
- Create signatures by drawing, typing, or uploading
- Place signatures anywhere on the document
- Save signatures for reuse

### Edit
- Add text boxes with formatting options
- Rotate, delete, or reorder pages
- Merge multiple PDFs into one
- Split PDFs into separate files

### Export & Share
- Save edited PDFs with annotations
- Export pages as images (PNG, JPEG)
- Print with customizable options
- Share via email or link
