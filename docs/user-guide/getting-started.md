# Getting Started with PaperFlow

This guide will walk you through your first steps with PaperFlow.

## Opening a PDF

### Method 1: Click to Select
1. Click the "Open PDF" button or the file drop area
2. Select a PDF file from your computer
3. The document will load in the viewer

### Method 2: Drag and Drop
1. Drag a PDF file from your file manager
2. Drop it onto the PaperFlow window
3. The document will load automatically

### Method 3: Install as PWA
1. Click the install button in your browser's address bar
2. PaperFlow will be installed as an app
3. Double-click PDF files to open them in PaperFlow

## Navigating Your Document

### Basic Navigation
- **Next/Previous Page**: Use the arrow buttons or arrow keys
- **Go to Page**: Click the page number and type a new one
- **Scroll**: Use mouse wheel or touchpad to scroll through pages

### Zoom Controls
- **Zoom In/Out**: Use the + and - buttons
- **Fit to Width**: Click "Fit Width" to match the viewer width
- **Fit to Page**: Click "Fit Page" to see the full page
- **Pinch to Zoom**: On touch devices, use two fingers to zoom

### Thumbnail View
1. Click the thumbnails icon in the sidebar
2. Click any thumbnail to jump to that page
3. Drag thumbnails to reorder pages

## Making Your First Annotation

### Adding a Highlight
1. Select the Highlight tool (press `H`)
2. Click and drag across text you want to highlight
3. Release to apply the highlight
4. Click the highlight to change its color

### Adding a Note
1. Select the Note tool (press `N`)
2. Click where you want to place the note
3. Type your comment in the popup
4. Click outside to close

### Drawing
1. Select the Draw tool from the toolbar
2. Choose a color and thickness
3. Draw directly on the document
4. Press `Esc` to exit drawing mode

## Saving Your Work

### Save to Your Computer
1. Click the Save button or press `Ctrl+S`
2. Choose a location to save the file
3. The PDF will be saved with all annotations

### Export Options
- **As PDF**: Keeps annotations editable
- **As Image**: Exports current page as PNG or JPEG
- **Flatten**: Bakes annotations into the PDF permanently
